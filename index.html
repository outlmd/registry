outl plugin registry

The day-zero "store" for outl plugins — a single versioned JSON index, no server, no API.

Clients read the index from this domain:

https://plugins.outl.app/registry.json

Install a plugin from its GitHub source:

outl plugin install github:user/repo
outl plugin install github:user/repo#v1.2.0
  • registry.json — the index
  • schema/registry-v1.json — the JSON Schema it validates against
  • github.com/outlmd/registry — open a PR to add yours
Served statically from outlmd/registry via Netlify.
